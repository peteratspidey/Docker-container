repo describes the steps to implement the docker containers and use it
## Containerisation 
* software containerisation - a method of packaging an application with its dependencies
                          - it allows applications to run consistently across different computing environments
* containers - a way to package and distribute applications
            - allows to create portable applications
            - useful for testing and debuggin applications 
* software required - 1. code - 
                    2. libraries
                    3. dependencies
                    4. runtime
